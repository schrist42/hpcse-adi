\subsection{Detailed Description}
Viewer for the Gray-\/\-Scott Reaction Diffusion simulation.

This class implements a viewer for the Gray-\/\-Scott Reaction Diffusion simulation. It visualizes the field U as the simulation is running using Open\-G\-L.
